# Nodejs API

 

## Installation

Use the package manager [npm](https://pip.pypa.io/en/stable/) to install .

```bash
npm install 
npm install jsonwebtoken --save
npm install bcryptjs --save
```

## Usage

nodemon server 
or
node server.js
Development server(Browser) - localhost:3000
```
