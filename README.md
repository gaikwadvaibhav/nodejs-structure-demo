# Nodejs API

 

## Installation

Use the package manager [npm](https://npmjs.com/) to install .

```bash
npm install 
npm install jsonwebtoken --save (Optional)
npm install bcryptjs --save (Optional)
```

## Usage
```bash
> nodemon server 
or
> node server.js

# Development server(Browser) - localhost:3000
```
